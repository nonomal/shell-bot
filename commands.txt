run - <command> 执行命令
enter - <text> 将文本输入到命令执行暂停需要交互的位置
type - <text> 将按键键入命令行
control - <a-z> 发送 Ctrl + a-z 到命令行
meta - (text) 给按键添加 Alt
keypad - 打开/关闭 特殊按键键盘
redraw - 强制命令重绘
end - 发送 EOF 到命令行
cancel - 中断命令执行
kill - 强制中断命令执行
status - 查看状态和当前设置
cd - 更改命令行工作目录
env - 更改环境变量
shell - 更改用于运行命令的 shell 程序
resize - 更改终端大小
setsilent - 启用/禁用 命令输出
setlinkpreviews - 启用/禁用 链接预览
setinteractive - 启用/禁用 shell 交互式标志
help - 获得帮助
file - 查看和编辑小型文本文件
upload - 上传和覆盖原始文件
r - run 或者 enter 的别名
